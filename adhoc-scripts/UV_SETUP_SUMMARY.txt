WHY UV?
================================

✓ 10-100x faster than pip
✓ Automatic Python version management
✓ Reproducible builds (uv.lock)
✓ Works like modern package managers (Cargo, npm)
✓ Replaces both pip and venv
✓ No more build system conflicts

================================
TROUBLESHOOTING
================================

Issue: Python 3.11 not found
  → UV will try to download it automatically
  → Or install: brew install python@3.11
  → Then: uv sync --python 3.11

Issue: "uv: command not found"
  → Install: pip install uv

Issue: Permission denied on .venv/bin/activate
  → Make executable: chmod +x .venv/bin/activate
  → Or use: uv run python main.py (no activation needed)

Issue: Dependency conflicts
  → Remove and resync:
     rm -rf .venv uv.lock
     uv sync

Issue: Hatchling build errors (FIXED)
  → Build system removed - no more packaging issues
  → Apps are now simple Python projects

================================
REFERENCE COMMANDS
================================

# Basic operations
uv sync                  # Create/sync environment
uv run python main.py    # Run without activation
uv pip list             # Show installed packages

# Advanced
uv sync --python 3.11   # Use specific Python version
uv sync --upgrade       # Upgrade all dependencies
uv pip install pkg      # Add a package
uv python list          # Show available Python versions
uv python install 3.11  # Download Python 3.11

# Environment activation (traditional way still works)
source .venv/bin/activate
pip list
deactivate
